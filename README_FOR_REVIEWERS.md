# Reviewer README

## Result

This package supports the fixed-case theorem

```text
[582, infinity) is contained in A_1({3,4,7}).
```

Here `A_1({3,4,7})` is the set of subset sums of distinct positive powers generated by `3`, `4`, and `7`.

## Scope

The submission proves only the fixed case `D={3,4,7}`, `k=1`.

No result is claimed for any other generating set.  
No result is claimed for `k != 1`.  
No optimality of the endpoint `582` is claimed.  
No density, asymptotic, or classification theorem is claimed.

## Proof structure

The proof uses:

1. a finite subset-sum certificate for `[582,4460]`;
2. an interval-overlap propagation lemma;
3. weighted next-gap identities;
4. a weighted gap lemma proved by a replayable modular certificate;
5. direct propagation to all later powers.

## Files

- `paper.tex` — journal-facing LaTeX source.
- `paper.pdf` — compiled PDF generated from `paper.tex`.
- `verify_initial_interval.py` — verifies the finite initial interval certificate.
- `verify_weighted_gap_certificates.py` — replays the weighted gap modular certificate.
- `WEIGHTED_GAP_CERTIFICATE_TABLES.md` — compressed modular certificate tables.
- `WEIGHTED_GAP_CERTIFICATE_REPLAY_PASS.log` — locally replayed weighted-gap pass output.
- `INITIAL_INTERVAL_REPLAY_PASS.log` — locally replayed initial-interval pass output.
- `PACKAGE_SHA256SUMS.txt` — SHA-256 checksums for the package files.

```bash

```
