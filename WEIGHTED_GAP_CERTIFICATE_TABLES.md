## 1. Purpose of this attachment

The direct interval-overlap proof requires the global overlap inequality:

\[
S(t)-t\ge1163
\]

for every future ordered power \(t>2187\) generated by \(3,4,7\), where

\[
S(t)=\sum_{\substack{p\in B\\p<t}}p,
\qquad
B=\{3^a:a\ge1\}\cup\{4^b:b\ge1\}\cup\{7^c:c\ge1\}.
\]

The weighted next-gap identities reduce this to the Weighted Gap Lemma.

This attachment records the full paired modular certificate structure used to close all weighted-danger systems.

---

## 2. Weighted Gap Lemma statement

For every future power \(t>2187\) generated by \(3,4,7\):

### Case A: \(t=3^r,\ r\ge8\)

\[
2\Delta_4+\Delta_7\ge7002.
\]

### Case B: \(t=4^s,\ s\ge6\)

\[
3\Delta_3+\Delta_7\ge7002.
\]

### Case C: \(t=7^q,\ q\ge4\)

\[
3\Delta_3+2\Delta_4\ge7002.
\]

The proof method is by finite full paired modular certificates.

# Full Paired Modular Test Definition

For a residue-admissible candidate \((u,v)\), a modulus \(Q\) eliminates the candidate when the full paired baseline system has no solution modulo \(Q\).

## Case A test

Given:

\[
a\equiv a_0\pmod{2187},
\]
\[
b\equiv b_0\pmod{2187},
\]

test whether there exists a baseline residue \(r\) such that

\[
4^a-3^r\equiv u\pmod Q,
\]
\[
7^b-3^r\equiv v\pmod Q.
\]

If no such \(r\)-class exists, the candidate is eliminated.

## Case B test

Given:

\[
a\equiv a_0\pmod{1024},
\]
\[
b\equiv b_0\pmod{512},
\]

test whether there exists a baseline residue \(s\) such that

\[
3^a-4^s\equiv u\pmod Q,
\]
\[
7^b-4^s\equiv v\pmod Q.
\]

If no such \(s\)-class exists, the candidate is eliminated.

## Case C test

Given:

\[
a\equiv a_0\pmod{2058},
\]
\[
b\equiv b_0\pmod{1029},
\]

test whether there exists a baseline residue \(q_0\) such that

\[
3^a-7^{q_0}\equiv u\pmod Q,
\]
\[
4^b-7^{q_0}\equiv v\pmod Q.
\]

If no such \(q_0\)-class exists, the candidate is eliminated.

# Reconstruct-and-Verify Code

The full row-level tables are too large for a compact markdown attachment. The following code reconstructs the residue-admissible sets and verifies certificate completion.

It verifies:

- interval certificate,
- residue-admissible candidates for Cases A/B/C,
- full paired modular eliminations,
- shadow examples,
- final completion status.

```python
# weighted_gap_certificate_tables_verify.py
#  — Weighted Gap Certificate Attachment Verification
# Scope: D = {3,4,7}, k = 1 only.
# No result for other generating sets is claimed.

from math import gcd
import hashlib


def order_mod(base, mod):
    if gcd(base, mod) != 1:
        return None
    x = 1
    for e in range(1, 10_000_000):
        x = (x * base) % mod
        if x == 1:
            return e
    raise RuntimeError(f"order not found for base={base}, mod={mod}")

def residue_log_map(base, mod):
    o = order_mod(base, mod)
    mp = {}
    x = 1
    for e in range(o):
        if x not in mp:
            mp[x] = e
        x = (x * base) % mod
    return o, mp

def values_for_forced_class(base, forced_mod, forced_res, ord_q, q):
    g = gcd(forced_mod, ord_q)
    start = forced_res % g
    return {pow(base, e, q) for e in range(start, ord_q, g)}

def full_pair_possible(case_label, u, v, a0, b0, q):
    if case_label == "A":
        base_a, base_b, base_x = 4, 7, 3
        forced_a_mod, forced_b_mod = 2187, 2187
    elif case_label == "B":
        base_a, base_b, base_x = 3, 7, 4
        forced_a_mod, forced_b_mod = 1024, 512
    elif case_label == "C":
        base_a, base_b, base_x = 3, 4, 7
        forced_a_mod, forced_b_mod = 2058, 1029
    else:
        raise ValueError(case_label)

    if gcd(base_a, q) != 1 or gcd(base_b, q) != 1 or gcd(base_x, q) != 1:
        return None

    ord_a = order_mod(base_a, q)
    ord_b = order_mod(base_b, q)
    ord_x = order_mod(base_x, q)

    avals = values_for_forced_class(base_a, forced_a_mod, a0, ord_a, q)
    bvals = values_for_forced_class(base_b, forced_b_mod, b0, ord_b, q)
    xvals = {pow(base_x, e, q) for e in range(ord_x)}

    target_from_a = {(A - u) % q for A in avals}
    target_from_b = {(B - v) % q for B in bvals}

    return bool(xvals & target_from_a & target_from_b)

def verify_interval_certificate():
    b_initial = [3,4,7,9,16,27,49,64,81,243,256,343,729,1024,2187]
    sums = {0}
    for x in b_initial:
        sums |= {s + x for s in list(sums)}
    return all(n in sums for n in range(582, 4461))

def build_case_A():
    _, map4 = residue_log_map(4, 6561)
    _, map7 = residue_log_map(7, 6561)
    out = []
    for u in range(1, 3501):
        max_v = 7001 - 2*u
        if max_v < 1:
            continue
        if u % 6561 not in map4:
            continue
        a0 = map4[u % 6561]
        for v in range(1, max_v + 1):
            if v % 6561 in map7:
                out.append(("A", u, v, a0, map7[v % 6561]))
    return out

def build_case_B():
    _, map3 = residue_log_map(3, 4096)
    _, map7 = residue_log_map(7, 4096)
    out = []
    for u in range(1, 2334):
        max_v = 7001 - 3*u
        if max_v < 1:
            continue
        if u % 4096 not in map3:
            continue
        a0 = map3[u % 4096]
        for v in range(1, max_v + 1):
            if v % 4096 in map7:
                out.append(("B", u, v, a0, map7[v % 4096]))
    return out

def build_case_C():
    _, map3 = residue_log_map(3, 2401)
    _, map4 = residue_log_map(4, 2401)
    out = []
    for u in range(1, 2334):
        max_v = (7001 - 3*u) // 2
        if max_v < 1:
            continue
        if u % 2401 not in map3:
            continue
        a0 = map3[u % 2401]
        for v in range(1, max_v + 1):
            if v % 2401 in map4:
                out.append(("C", u, v, a0, map4[v % 2401]))
    return out

def eliminate_with_moduli(candidates, moduli):
    rows = []
    unresolved = list(candidates)
    counts = []

    for q in moduli:
        next_unresolved = []
        eliminated = 0

        for row in unresolved:
            case_label, u, v, a0, b0 = row
            possible = full_pair_possible(case_label, u, v, a0, b0, q)
            if possible is False:
                rows.append((*row, q))
                eliminated += 1
            else:
                next_unresolved.append(row)

        unresolved = next_unresolved
        counts.append((q, eliminated, len(unresolved)))

        if not unresolved:
            break

    return rows, unresolved, counts

def sha256_rows(rows):
    h = hashlib.sha256()
    for row in rows:
        h.update(repr(row).encode("utf-8"))
        h.update(b"\n")
    return h.hexdigest()

def main():
    interval_ok = verify_interval_certificate()

    case_A = build_case_A()
    case_B = build_case_B()
    case_C = build_case_C()

    expected_counts_ok = (
        len(case_A) == 1_361_889 and
        len(case_B) == 255_938 and
        len(case_C) == 1_501_666
    )

    moduli_A = [5,13,19,31,37,41,43,73,109,487,2917,17497]
    moduli_B = [5,11,13,17,19,29,31,37,41,43,257]
    moduli_C = [5,13,17,19,29,31,37,41,43,73,197,1373]

    rows_A, un_A, counts_A = eliminate_with_moduli(case_A, moduli_A)
    rows_B, un_B, counts_B = eliminate_with_moduli(case_B, moduli_B)
    rows_C, un_C, counts_C = eliminate_with_moduli(case_C, moduli_C)

    all_rows = rows_A + rows_B + rows_C
    cert_hash = sha256_rows(all_rows)

    shadow_A = ("A", 4, 7, 1, 1)
    shadow_B = ("B", 3, 7, 1, 1)
    shadow_C = ("C", 3, 4, 1, 1)

    shadow_A_elim = full_pair_possible(*shadow_A, q=13) is False
    shadow_B_elim = full_pair_possible(*shadow_B, q=5) is False
    shadow_C_elim = full_pair_possible(*shadow_C, q=5) is False

    print("interval_ok =", interval_ok)
    print("case_A_residue_admissible =", len(case_A))
    print("case_B_residue_admissible =", len(case_B))
    print("case_C_residue_admissible =", len(case_C))
    print("expected_counts_ok =", expected_counts_ok)

    print("case_A_counts =", counts_A)
    print("case_B_counts =", counts_B)
    print("case_C_counts =", counts_C)

    print("case_A_unresolved =", len(un_A))
    print("case_B_unresolved =", len(un_B))
    print("case_C_unresolved =", len(un_C))

    print("shadow_A_eliminated =", shadow_A_elim)
    print("shadow_B_eliminated =", shadow_B_elim)
    print("shadow_C_eliminated =", shadow_C_elim)

    print("certificate_row_count =", len(all_rows))
    print("certificate_sha256 =", cert_hash)

    if (
        interval_ok and expected_counts_ok and
        not un_A and not un_B and not un_C and
        shadow_A_elim and shadow_B_elim and shadow_C_elim
    ):
        print("WEIGHTED_GAP_PROVED")
    else:
        print("WEIGHTED_GAP_INCOMPLETE")

if __name__ == "__main__":
    main()
```
